# Kotoba Note

一个更贴近真实学习节奏的日语单词手帐应用。

`Kotoba Note` 用 `PySide6 + SQLite + Janome` 构建，围绕课文拆词、词汇整理、复习、自测、错题回看和学习报告，尽量把“记单词”做成更顺手、更有人味的长期学习流程。

## 主要功能

- 课文手札：导入日语文本，自动分析重点词、词频和注音
- 词汇本：支持筛选、标签、优先级、批量整理和回收恢复
- 复习计划：按学习节奏安排每日复习，支持多方向练习
- 随机自测：提供即时反馈，答题后可停留回看
- 错题本：追踪高频失误词，支持专项练习和回流复习
- 学习报告：查看趋势、分布和下一步建议

## 启动方式

```bash
pip install -r requirements.txt
python main.py
```

## 项目结构

```text
app/

  ui/          主界面、导航、组件、样式
main.py        程序入口

```
